# research/

Esta pasta contem **artefatos de referencia transversal** — fichas de especialidades ja entregues que servem de referencia para cross-aprendizado e auditorias.

## O que fica aqui

- Auditorias de qualidade de JSONs de fichas ja entregues
- Analises comparativas entre especialidades
- Qualquer artefato de pesquisa que nao pertence a uma especialidade especifica

## O que NAO fica aqui

Bancos de evidencias, playbooks e relatorios de sesssao **especificos de uma especialidade** ficam em `especialidades/{nome}/research/`.
